Import from Instapaper to Readability

This will import your Instapaper articles to Readability.

READ THIS FIRST

You will need to authorize 'Miguel Javier' to connect to you Readability account, and export your Instapaper articles as CSV. I don't keep any of your files or information.

Or you can get the code here to upload and run on your own server. You will need to have your own Readability API key to run it though.

In both cases, no warranty or support is guaranteed.

Instapaper only exports your last 2000 article URLs, and Readability will try to retrieve those articles from those URLs. If the website is no longer around, Readability won't be able to import the aritcle.
